Introduction to Python

Learn the basics of the world's fastest growing and most popular programming language used by software engineers, analysts, data scientists, and machine learning engineers alike.

Course Content

Week 1

Getting Started with Python

Learn the basics of Python 3 syntax. Setup our environments and learn about basic programming concepts.

Week 2

Sequences and Iteration

Learn about Python's list structure to store sequential data. Learn about loops and how to iterate through lists.

Week 3

Text Manipulation

Learn how to handle text with Pythons string methods. Will practice parsing and how to deal with strings in Python.

Week 4

Object Oriented Programming

Learn about functions and how to organize data into classes.

Week 5

I/O and Data Structure Introduction

Learn about file handling and how to handle user input. Will learn about Python dictionaries and how to apply them.

Week 6

Final Project Lab

Build a piece of software in a lab style environment from start to finish.

FAQs

  • Why Python (and not Java)?

    Python is less verbose than Java and requires very little setup to get running. Python is robust enough that it can be used to quickly write scripts, create servers, and other things you may want to use it for without having to do a lot of setup.

  • What previous experience do I need for this class?

    This class is designed for those who have little to no programming experience and want to get into programming. We strive to establish a solid foundation in CS so that our students can apply what they have learned.

  • What will I be able to create by the end of the class?

    We will blend all the concepts that we learn into a cohesive final project that can be presented. The material covered in the final project can be used as a baseline for other personal projects.
