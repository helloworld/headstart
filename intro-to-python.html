Course Content

**Getting Started with Python 3** Learn the basics of Python 3 syntax. Setup our environments and learn about basic programming concepts.

Week 1

**Sequences and Iteration** Learn about Python's list structure to store sequential data. Learn about loops and how to iterate through lists.

Week 2

**Text Manipulation** Learn how to handle text with Pythons string methods. Will practice parsing and how to deal with strings in Python.

Week 3

**Object Oriented Programming** Learn about functions and how to organize data into classes.

Week 4

**I/O and Data Structure Introduction** Learn about file handling and how to handle user input. Will learn about Python dictionaries and how to apply them.

Week 5

**Final Project Lab** Build a piece of software in a lab style environment from start to finish.
